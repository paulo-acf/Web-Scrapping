## **Miniprojeto Educacional (Web Scraping)**

## 

### O objetivo deste trabalho é desenvolver um miniprojeto pessoal que demonstre uma forma de realizar Web Scraping.

### 

### Para tornar este miniprojeto mais didático, achou-se conveniente acrescentar comentários junto ao código (isto facilitará a revisăo/estudo futuro do referido trabalho).

### 

### **# # # # # # #**

### 

### Este projeto utiliza as dependęncias listadas em requirements.txt

### 

### Para clonar o repositório e criar um ambiente virtual: python -m venv venv

### 

### Para ativar o ambiente virtual:

### • Windows: venv\\Scripts\\activate

### • Linux / macOS: source venv/bin/activate

### 

### Instale as dependęncias com: pip install -r requirements.txt

### 

### Execute o script com: python script.py
